Events and Tournaments

Events for PLAN 29 are currently as follows.
If you have any questions or suggestions please email Computer Club RSO Email.

Other rooms are available. Start your own tourney, game of cards against humanity, or watch a movie.


IMPORTANT: Please sign in before bringing in systems. That way you'll know where you're going.


IMPORTANT: Tournaments will start when scheduled. Please arrive early to make sure you're signed in!

NOTE:Tourment end times are not set in stone. Tournaments are likely to end earlier.

Sign-in and concessions will be located in the first floor entryway.
Different tournaments are in different rooms.


Many tournaments run at the same time. Please consult the FAQ section of the About page if you're interested in more than 1.
